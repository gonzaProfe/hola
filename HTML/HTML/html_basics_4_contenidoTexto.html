Conclusiones:

  • Hay elementos para seccionar el contenido de nuestro documento, es decir, agrupamos elementos HTML con otros elementos HTML para mejorar la accesibilidad de nuestro proyecto
  • Hay elemetnos que permiten establecer un contenido de texto. Por ejemplo, establecer una lista, un párrafo, una cita, etc
  • main: elemento que especifica cual es el contenido principal del body
  • section: elemento que representa una sección genérica del documento
  • article: como su nombre lo indica es un artículo. Podría representar: un producto en un ecommerce, una noticia en un diario virtual, un artículo en un blog, etc
  • ul: lista desordenada. Permite establecer un elemento lista en la cual sus items pueden estar ordenados de cualquier forma.
  • li: elemento de lista. Establece un item de la lista
  • footer: es el pie de página
